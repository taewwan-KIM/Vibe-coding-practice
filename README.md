# 오목 · AI 대전

> AI와 1:1로 대국하는 싱글 페이지 오목 웹 서비스

![HTML](https://img.shields.io/badge/HTML-단일파일-orange) ![Vanilla JS](https://img.shields.io/badge/JS-Vanilla-yellow) ![No Build](https://img.shields.io/badge/빌드-불필요-green)

## 플레이

`index.html` 파일을 브라우저에서 열면 바로 플레이할 수 있습니다. 별도 설치나 서버 불필요.

## 주요 기능

### 게임 설정
| 옵션 | 선택지 |
|------|--------|
| 선공 | 플레이어(●흑) / AI(●흑) |
| AI 난이도 | 쉬움 / 보통 / 어려움 |
| 착수 제한 시간 | 30초 / 1분(기본) / 2분 |

### 오목 룰 (자동 적용)
- **승리 조건** — 흑: 정확히 5목 / 백: 5목 이상
- **금수 (흑 전용)** — 장목(6목↑), 사사(쌍사), 삼삼(쌍삼) 자동 차단
- **시간 초과** — 제한 시간 내 착수 없으면 자동 패

### AI 전략
| 난이도 | 전략 |
|--------|------|
| 쉬움 | 기존 돌 주변 랜덤 배치 |
| 보통 | 즉시 승리 → 상대 막기 → 위협 점수 평가 |
| 어려움 | 열린 4목 생성·차단 → 다방향 위협 분석 + 중앙 선호 |

## 기술 스택

- **HTML / CSS / Vanilla JS** — 외부 의존성 없음
- **Canvas API** — 19×19 바둑판 및 돌 렌더링 (그라디언트 효과)
- 단일 파일(`index.html`) 구조

## 프로젝트 배경

바이브코딩 첫 번째 프로젝트. 기획(`plan.md`) → 구현 → 완성의 전체 흐름을 빠르게 경험하는 것이 목표.
