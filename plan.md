# Vibe Coding – One Page Omok

## 개요
이 프로젝트는 단일 페이지에서 AI와 오목을 즐길 수 있는 웹 서비스다.
간단하지만 완성도 있는 인터랙션을 제공하며, 실제 서비스처럼 자연스럽게 플레이 가능한 경험을 목표로 한다.

## 핵심 컨셉
* 페이지 이동 없이 모든 플레이가 가능한 **One Page Web Service**
* 유저 또는 AI 먼저 착수하는 것을 선택하는 구조의 **AI 대전 오목**
* 빠른 템포와 긴장감을 위해 시간제한을 선택할수 있는 **시간 제한 규칙** 
 
## 주요 기능
1. **AI 오목 대전**
   * 사용자는 AI와 1:1로 오목을 플레이할 수 있다.
   * 착수의 선택은 유저가 선택(유저 or AI)한 이후 진행한다.

2. **첫 수 제한 시간**
   * 게임 시작 시 시간 설정을 하지 않으면 1분으로 자동으로 제한된다.
   * 시간은 최소 30초에서 최대 2분으로 선택이 가능하다.
   * 불필요한 대기 상태를 방지하고 플레이 흐름을 유지한다.

3. **AI 난이도 선택**
   * 게임 시작 전 AI의 난이도를 선택할 수 있다.
   * 난이도에 따라 AI의 수 계산 깊이와 전략이 달라진다.

4. **자동 적용되는 오목 룰**
   * 오목의 승패 판정 및 금수 여부 등은 서비스에서 미리 정의된 규칙에 따라 자동 처리된다.
   * 사용자는 별도의 룰 설정 없이 바로 게임에 집중할 수 있다.

## UX / UI 방향
* 복잡한 메뉴 없이 직관적인 인터페이스
* 게임 상태(차례, 제한 시간, 승패)를 한눈에 확인 가능
* 불필요한 설명을 줄이고 플레이 중심의 화면 구성

## 목표
* “연습용”이 아닌, 실제로 서비스처럼 보이고 동작하는 웹 서비스 구현
* 기획 → 구현 → 완성의 전체 흐름을 빠르게 경험
* 바이브코딩의 첫 프로젝트로서 기준이 되는 결과물 제작
